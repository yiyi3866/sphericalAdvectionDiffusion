# advectionDiffusionSphe

**************************************************************************
************* advectionDiffusionSphe (aDS) for OpenFOAM ********************
**************************************************************************

* General Information

Solving thermal advection-diffusion equation in spherical systems using OpenFOAM.
Can apply to ballooning intrusions with ductile-deformed surrounding rocks (aureole width as an input).
The package includes for now:

    - a "spheAdvDiff" solver
    - an exampleCase


* This toolbox has been tested on OpenFOAM v11 only (use the v11legacy branch)
OpenFOAM installation: www.openfoam.org
